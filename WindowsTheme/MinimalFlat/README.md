# Minimal Flat Dark - Windows 11 Theme

Тёмная минималистичная тема для Windows 11 с акцентами цвета Teal.

## Установка

### Быстрая установка
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\Install.ps1
```

### Ручная установка
1. Запустите `MinimalFlat.reg` и подтвердите импорт
2. Перейдите в **Настройки > Персонализация > Темы**
3. Выберите тему "Minimal Flat Dark"

## Требования

- Windows 11 Build 22000+
- (Опционально) [UltraUXThemePatcher](https://mhoefs.eu/index.php?option=com_content&view=article&id=45:ultrauxthemepatcher-for-windows-10-8-1-8-7-vista-xp&catid=10&Itemid=104) для кастомных .msstyles

## Удаление

```powershell
.\Install.ps1 -Uninstall
```

Или через **Настройки > Персонализация > Темы** выберите стандартную тему.
