**PiCodePreviewComponent** (~140 lines):
- Standalone + OnPush + signal-based.
- Tree-shake imports: highlight.js/lib/core + 5 languages (bash/css/json/typescript/xml).
- 'xml' aliased to 'html' (hljs convention).
- **Single registration** via `afterNextRender({ once: true })` —
  languages registered ONCE per app load (not per component instance),
  avoids 5× define() calls per page mount.
- SSR-safe: hljs.highlight() is sync + DOM-free; afterNextRender never
  fires on server; highlight() with unregistered language falls back to
  escapeHtml() (prevents XSS via [innerHTML] binding).
- Inputs: code (required), language (html|typescript|css|bash|json),
  ariaLabel, showLineNumbers (default true), showCaption (default true),
  fileName (optional), **captionTitle (new — custom caption override)**.
- Computed highlightedCode → safe HTML from hljs.highlight() with
  ignoreIllegals: true. Fallback to escapeHtml for unregistered langs.
- Computed lineNumbers → array of 1..N for sidebar gutter.
- Template: <figure> + optional figcaption (captionTitle ?? language() +
  fileName) + flex layout (line-numbers sidebar + highlighted <pre><code>).
- **Paper & Ink highlight.js theme SCOPED inside component** via `:host ::ng-deep`
  selectors — keywords → --color-accent-cool, strings → --color-accent-warm,
  comments → --color-muted (italic), titles → --color-ink bold,
  numbers → --color-ink, deletions → --color-destructive. NO #hex, NO box-shadow.

**CodePreviewPage** (~85 lines, NEW):
