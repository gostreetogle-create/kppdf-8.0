═══════════════════════════════════════════════════════════════

1. Текущий код: <app-ui-page-header>, <mat-card> filter/error/table,
   <app-ui-badge> для status indicators, mat-icon-button actions.

2. Проблемы:
   - mat-card вместо ui-card.
   - "Системную единицу нельзя удалить" inline `<button mat-icon-button
     color="warn" [disabled]="u.isSystem">` — лучше через
     <ui-icon-btn variant="danger" [disabled]="u.isSystem">.

3. Контекст: badge already migrated to <app-ui-badge>; только
   filter/error/table card нужно migrate.

═══════════════════════════════════════════════════════════════
ЧТО ДЕЛАТЬ
═══════════════════════════════════════════════════════════════

ШАГ 1: Filter card → <ui-card variant="filter">.
ШАГ 2: Error card → <ui-alert variant="error"> (или оставить mat-card).
ШАГ 3: Table card → <ui-card variant="table">.
ШАГ 4: Create button → <ui-btn variant="primary" (onClick).
ШАГ 5: Reload button → <ui-btn variant="ghost">.
ШАГ 6: Search clear → <ui-icon-btn>.
ШАГ 7: Action buttons в row-actions cell:
  - `<button mat-icon-button matTooltip="Редактировать">` →
    `<button mat-icon-button>` (без изменений, или обернуть в
    <app-ui-table-row-actions>).
  - `<button mat-icon-button matTooltip="Удалить" color="warn"
    [disabled]="u.isSystem">` →
    `<button mat-icon-button color="warn" [disabled]="u.isSystem">`.

ШАГ 8: Удалить лишние импорты.
