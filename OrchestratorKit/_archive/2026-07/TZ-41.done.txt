ИСХОДНОЕ СОСТОЯНИЕ
════════════════════════════════════════════════════════

1. Текущий kppdf-8.0 frontend использует Angular Material
   (`mat-flat-button`, `mat-card`, `mat-input` и т.д.) —
   SaaS-look со shadow, rounded-3xl и gradient fills.
2. Paper & Ink требует hand-rolled примитив с hairline 1px
   border + OKLCH tokens из TZ-32 + signal inputs/outputs API
   (`input()` / `output()`, НЕ `@Input` / `@Output` decorator'ы).
3. Директория `frontend/src/app/shared/ui/select/` пуста или со-
   держит deprecated stub из ранних TZ — в этой TZ удалить и
   заменить на новый standalone OnPush signal-based компонент.
ЧТО ДЕЛАТЬ
═══════════════════════════════════════════════════════════════

1. Реализация на `@angular/cdk/listbox` (headless, a11y-ready).
   Три компонента: `<app-pi-select>` (host, signal value),
   `<app-pi-select-trigger>` (button), `<app-pi-select-option>`
   (item). Использовать CdkListbox + CdkOption из CDK.

2. signal inputs на app-pi-select:
   - value: model<string|null>(null),
   - placeholder: 'Выберите значение' (rus default),
   - disabled: boolean = false,
   - size: 'sm'|'md' (default 'md').

3. Outputs:
   - valueChange = output<string|null>().

4. Стили hairline: trigger = bg-paper border-rule, options multi-row
   bg-paper hairline-b между, hover: bg-paper-2.
   Selected option: bg-ink text-paper.
   НЕТ shadows. NO rounded-3xl.
