ДИАЛОГОВОЕ ОКНО «НОВЫЙ ШАБЛОН ТАБЛИЦЫ» — ПРОМПТ ДЛЯ ИИ

---

Общие требования:
- Окно на всю ширину экрана (90-95%), без «дёргания» при взаимодействии с элементами
- Фиксированная высота без вертикального скролла внутри формы
- Горизонтальное разделение: верхняя часть — настройки, нижняя часть — таблица

---

ВЕРХНЯЯ ЧАСТЬ (настройки):

1. Выбор типа создания:
   - Табы или кнопки-переключатели: «Новая таблица» | «Из существующих данных» | «Дубликат шаблона»

2. Если «Новая таблица»:
   - Название * (обязательное поле)
   - Описание (текстовое поле)
   - Категория: выбор из списка (Спецификация, Калькуляция, Сводка заказа, Прайс-лист, Прочее)
   - Источник данных: выпадающий список
   - Порядок: числовое поле
   - Активен: чекбокс

3. Если «Из существующих данных»:
   - Выбор таблицы-источника (выпадающий список)
   - Выбор полей для отображения (мультиселект или список с чекбоксами)
   - Автоматическое заполнение столбцов на основе выбранных полей

4. Если «Дубликат шаблона»:
   - Выбор существующего шаблона из списка (выпадающий список с поиском)
   - Все настройки копируются, название получает суффикс «(копия)»
   - Пользователь сразу может редактировать скопированный шаблон

---

НИЖНЯЯ ЧАСТЬ (таблица настроек столбцов):

Заголовок секции: «Настройка столбцов»

Колонки таблицы:
1. № — порядковый номер (авто, не редактируется)
2. Название столбца — текстовое поле (обязательное)
3. Тип данных — выпадающий список: Текст, Число, Дата, Валюта, Булево
4. Ширина — числовое поле + ползунок (px или %) + возможность перетаскивания границы столбца мышкой
5. Выравнивание — кнопки: лево | центр | право
6. Формат — выпадающий список (зависит от типа: для числа — кол-во знаков после запятой, для даты — формат, для валюты — символ)
7. Обязательный — чекбокс
8. Заголовок — текстовое поле (отображаемое имя в шапке итоговой таблицы)
9. Действия — кнопки: вверх ↓, вниз ↑, удалить 🗑

Дополнительно:
- Кнопка «+ Добавить столбец» (внизу таблицы)
- Drag & drop строк для изменения порядка
- Визуальный индикатор ширины столбца (пропорциональная полоска)

---

ПРЕДПРОСМОТР (справа, 40% ширины):
- Живой preview итоговой таблицы с реальными данными
- Кнопка «Обновить preview»
- Столбцы отражают настройки в реальном времени

---

НИЖНЯЯ ПАНЕЛЬ (футер):
- Кнопки: «Отмена» ( outlined ), «Сохранить» ( primary )
- Справа: индикатор состояния (заполнено/всего обязательных полей)

---

ДОПОЛНИТЕЛЬНЫЕ ФУНКЦИИ:
- Дублирование шаблона из списка существующих
- Валидация: подсветка ошибок прямо в таблице настроек
- История изменений (отмена последнего действия)
